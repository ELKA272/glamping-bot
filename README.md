# 🤖 Telegram-бот для бронирования глэмпинга «уДачная территория»

## Быстрый запуск

```bash
cd ~/Desktop/glamping-site/bot
pip3 install -r requirements.txt
```

## Настройка

1. Откройте Telegram, найдите **@BotFather** и отправьте команду `/newbot`
2. Введите имя бота (например, `уДачная территория бот`)
3. Введите username (например, `Udachnaya_bot`)
4. Скопируйте полученный токен (строка вида `1234567890:ABCdefGHIjkl...`)
5. Откройте `bot.py` и замените `ВАШ_ТОКЕН_ОТ_BOTFATHER` на полученный токен
6. Узнайте ваш `ADMIN_CHAT_ID`:
   - Запустите бота: `python3 bot.py`
   - Напишите боту `/start`
   - Затем напишите любому боту, который показывает ID (например, `@userinfobot`)
   - Вставьте ID в `ADMIN_CHAT_ID` в `bot.py`

## Запуск

```bash
python3 bot.py
```

## Для постоянной работы (хостинг)

Рекомендую запустить на бесплатном хостинге:
- **Railway.app** — бесплатно, просто залить этот код

## Команды бота

| Команда | Описание |
|---------|----------|
| `/start` | Приветствие и главное меню |
| `/book` | Начать бронирование |
| `/houses` | Список домиков и цены |
| `/prices` | Все цены |
| `/faq` | Частые вопросы |
| `/contacts` | Контакты |
| `/bookings` | Все заявки (только для админа) |

## Что делает бот

- Показывает домики с ценами
- Собирает даты заезда/выезда
- Автоматически считает стоимость (будни дешевле)
- Предлагает добавить баню
- Сохраняет заявку в `bookings.json`
- Отправляет уведомление владельцу в Telegram
